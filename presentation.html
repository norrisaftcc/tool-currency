TOOL-CURRENCY PROJECT

A retro-themed currency converter with nostalgic 80s terminal interface

Presented by Student A & Student B

Project Overview

Tool-Currency is a Streamlit GUI application that allows users to convert between a dozen major global currencies. The application features a retro-styled terminal interface reminiscent of 80s computing systems, complete with glowing green text, scanline effects, and a nostalgic aesthetic.

API Integration

I implemented integration with the Open Exchange Rates API to provide real-time exchange rates:

# Real-time exchange rate fetching
def get_exchange_rates(base_currency="USD"):
    try:
        response = requests.get(f"https://open.er-api.com/v6/latest/{base_currency}")
        data = response.json()
        
        if data.get('result') == 'success':
            return data['rates'], data['time_last_update_utc']
        else:
            # Fallback to sample data if API call fails
            return get_sample_exchange_rates(base_currency), datetime.now().strftime(...)
    except Exception:
        # Fallback to sample data if API call fails
        return get_sample_exchange_rates(base_currency), datetime.now().strftime(...)

Fallback Mechanism

I created a robust fallback system to ensure the application works even without internet connectivity:

# Sample exchange rates as a fallback
def get_sample_exchange_rates(base_currency="USD"):
    # Sample exchange rates (as of May 2025)
    base_rates = {
        "USD": 1.0,
        "EUR": 0.93,
        "GBP": 0.79,
        "JPY": 154.56,
        # Additional currencies...
    }
    
    # Recalculate all rates if base currency is not USD
    if base_currency != "USD":
        base_value = base_rates[base_currency]
        converted_rates = {}
        
        for currency, rate in base_rates.items():
            converted_rates[currency] = rate / base_value
            
        return converted_rates
    
    return base_rates

Conversion History Tracking

I implemented a history tracking system to record all currency conversions:

# Initialize session state for conversion history
if "conversion_history" not in st.session_state:
    st.session_state.conversion_history = []

# Add to conversion history when a conversion is performed
timestamp = datetime.now().strftime("%Y-%m-%d %H:%M:%S")
st.session_state.conversion_history.append({
    "timestamp": timestamp,
    "from_currency": from_currency,
    "to_currency": to_currency,
    "amount": amount,
    "result": result
})

# Display conversion history
if len(st.session_state.conversion_history) > 0:

CONVERSION HISTORY
